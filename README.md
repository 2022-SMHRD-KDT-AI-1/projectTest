## 프로젝트명
지도 API를 활용한 화장실 안내 서비스
## 주요기능
- 지도기반의 각 지역별 화장실 안내
- 화장실 청결도 점수관리
- 회원관리 기능(로그인,회원가입,탈퇴)
- 커뮤니티 게시판
## 개발환경
- 사용언어 : Java 1.8 / HTML / CSS / Javascript
- 개발도구 : eclipse / VSCode
- 서버환경 : tocatm 8.5 / Flask
- 데이터베이스 : oracle 11g
- 빌드도구 : maven
- 디자인도구 : PhotoShop
